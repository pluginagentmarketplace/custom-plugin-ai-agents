## Common Programming Patterns

### Error Handling
```python
# Python - EAFP (Easier to Ask for Forgiveness than Permission)
try:
    result = risky_operation()
except ValueError as e:
    handle_error(e)
```

```go
// Go - Explicit error returns
result, err := riskyOperation()
if err != nil {
    return fmt.Errorf("operation failed: %w", err)
}
```

```rust
// Rust - Result type
let result = risky_operation()?; // Propagate error
```

### Concurrency
```go
// Go - Goroutines and channels
ch := make(chan int)
go func() { ch <- compute() }()
result := <-ch
```

```rust
// Rust - async/await with Tokio
let result = tokio::spawn(async { compute().await }).await?;
```

```python
# Python - asyncio
result = await asyncio.gather(*[compute() for _ in range(10)])
```
